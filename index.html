2-Day Schedule — What you’ll actually learn

Day 1 — AI Tools & Productivity

  • Tool selection: which free AI tools solve which developer tasks (code completion, linting, refactor, docs).
  • Debug faster: using AI to triage failing tests, interpret stack traces and craft reproducible bug reports.
  • Prompts & templates: reproducible prompt patterns for code generation, reviews, and documentation.
  • Workflow integration: plug AI tools into your IDE, pipelines and PR workflows safely.

Day 2 — Vibe Code

  • AI-assisted development: generate scaffold, wire up APIs and tests with focused prompts.
  • Maintainable patterns: how to prompt for readable, testable code and avoid hallucinations.
  • Mini-build: guided hands-on session — ship a small feature using AI + review.
  • Q&A & next steps: tailored suggestions for your projects and learning path.
